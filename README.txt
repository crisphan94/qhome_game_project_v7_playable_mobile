HÓA THÂN VÀO NGHỀ XÂY DỰNG - GODOT V7

Nội dung:
- Splash + logo dùng ảnh bạn cung cấp
- Menu chính
- 2 scene chơi được: office và site
- Nhân vật di chuyển bằng bàn phím hoặc joystick cảm ứng
- Nút Tương tác cho điện thoại
- NPC hội thoại + lựa chọn
- Kết quả dự án
- Save/load
- Âm thanh placeholder WAV

Cách mở:
1. Cài Godot 4.2 hoặc mới hơn
2. Import thư mục này bằng file project.godot
3. Run project

Cách build APK:
1. Cài Android SDK + JDK
2. Trong Godot, cài Android export templates
3. Project > Export > Add Android
4. Chọn export ra APK

Lưu ý:
- Đây là prototype để tiếp tục mở rộng.
- Bạn có thể thay asset hình, âm thanh, font sau.
